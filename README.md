# Portfólio Pessoal – Introdução à Programação Front-End e Back-End

O objetivo será aplicar os conhecimentos adquiridos ao longo do curso SCTEC, Introdução à Programação Front-End e Back-End, utilizando as tecnologias HTML, CSS e JavaScript para construir uma página estruturada, estilizada e com elementos interativos.

Conforme aprendemos no curso, a página está sendo organizada em seções distintas, incluindo uma apresentação pessoal, habilidades técnicas e comportamentais. A estrutura foi construída com HTML semântico, garantindo organização adequada do conteúdo. O design foi implementado utilizando CSS para estilização visual, enquanto o JavaScript foi empregado para adicionar interatividade, demonstrando compreensão de manipulação do DOM.

Seguindo as boas práticas de desenvolvimento, este projeto foi estruturado em pastas separadas para organização do código. Todos os arquivos necessários para execução estão incluídos neste pacote compactado. 

Para visualizar a aplicação, basta abrir o arquivo index.html em qualquer navegador como o Chrome 147+, Firefox 147+, Edge, Safari e/ou Opera One(R2).

Ao realizar esta atividade, consegui colocar em prática conceitos fundamentais de desenvolvimento web visto ao longo do início de Fevereiro/26, posso destacar a estruturação de páginas, estilização e programação básica no front-end.
